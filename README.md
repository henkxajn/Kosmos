# Pastel Orbits — 2D Top-Down (Inspector / No Build)

- Kliknięcie **preferuje planety** (żeby nie łapało głównie księżyców).
- Widzisz **nazwy planet i gwiazdy** (księżyce tylko po zbliżeniu albo gdy zaznaczone).
- Jest **Inspector** z informacjami o wybranym ciele (masa, promień, g, orbit).
- Lista wyboru ma podział: Gwiazda / Planety / Księżyce.
- Zoom max podniesiony, żeby dało się zejść do skali księżyców.

## GitHub Pages
Settings → Pages → main + /(root).
Wrzuć pliki do repo (root), nadpisz i odśwież stronę (czasem Ctrl+F5).
